# Football Fixtures
Football Fixtures is a simple example for modern android applications.

## Guideline
Football Fixture uses a static feed which is available in this [URL](https://run.mocky.io/v3/7e435a1a-86bb-43b8-ba48-678f5e7fd946). You can find it in feed.json too.

## Code Pattern
This code illustrates the use of MVVM architecture, Dagger dependency injection library, and using Retrofit And Rxjava together. 
Other libraries or Techniques: Nested Recyclerview, LiveData, ViewBinding, Glide, and SwipeRefreshLayout
